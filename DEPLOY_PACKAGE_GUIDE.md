# 小程序部署包下载 - 快速指南

## 🎯 如何获取部署包

### 步骤 1: 打开页面

在微信开发者工具的调试器中输入：

```javascript
wx.navigateTo({
  url: '/pages/deploy-download/index'
})
```

### 步骤 2: 生成部署包

点击页面上的 **「生成部署包」** 按钮

### 步骤 3: 下载文件

- 点击 **「复制链接到浏览器」** 按钮
- 在浏览器中粘贴链接
- 下载 `miniprogram-deploy-v1.0.0.tar.gz` 文件

### 步骤 4: 解压文件

```bash
tar -xzf miniprogram-deploy-v1.0.0.tar.gz
```

解压后会得到 `dist-weapp` 目录。

### 步骤 5: 在微信开发者工具中部署

1. 打开微信开发者工具
2. 点击 **「导入项目」**
3. 选择 `dist-weapp` 目录
4. 输入 AppID：`wxeb1d51afc9237cda`
5. 项目名称：宿舍管理助手
6. 点击 **「导入」**

## ✅ 完成！

导入后可以直接：
- 点击 **「预览」** 生成二维码，手机扫码测试
- 点击 **「上传」** 上传到微信小程序平台
- 点击 **「真机调试」** 在真机上调试
