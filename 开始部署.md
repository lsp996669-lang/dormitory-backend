### 第三步：配置并部署

**1. 配置环境变量**

在"版本配置"页面，添加以下环境变量：

```bash
# 必需配置
SUPABASE_URL=https://你的项目ID.supabase.co
SUPABASE_ANON_KEY=你的匿名密钥

# 运行环境
NODE_ENV=production
PORT=3000
```

**如何获取 Supabase 凭证：**
```
1. 登录 https://supabase.com
2. 进入你的项目
3. 点击左侧"设置" → "API"
4. 复制:
   - Project URL → SUPABASE_URL
   - anon public → SUPABASE_ANON_KEY
```

**2. 配置服务规格**
```
CPU: 0.5核
内存: 512MB
实例: 最小1个，最大2个
端口: 3000
```

**3. 点击"部署"按钮**
- 等待构建完成（约3-5分钟）
- 查看构建日志
- 确认部署成功
