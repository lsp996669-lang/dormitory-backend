# 下载小程序代码包 - 快速指南

## 🎯 如何使用

### 步骤 1: 打开页面

在微信开发者工具的调试器中输入：

```javascript
wx.navigateTo({
  url: '/pages/download-app/index'
})
```

### 步骤 2: 生成下载链接

点击页面上的 **「生成下载链接」** 按钮

### 步骤 3: 下载文件

- 点击 **「复制链接到浏览器」** 按钮
- 在浏览器中粘贴链接
- 下载 `miniprogram-v1.0.0.tar.gz` 文件

### 步骤 4: 解压文件

```bash
tar -xzf miniprogram-v1.0.0.tar.gz
```

解压后会得到 `dist-weapp` 目录，里面是完整的小程序代码。

### 步骤 5: 用微信开发者工具打开

1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择 `dist-weapp` 目录
4. 输入 AppID：`wxeb1d51afc9237cda`
5. 点击导入
