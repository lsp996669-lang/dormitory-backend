# 🚀 微信云开发部署指南

## 📋 部署流程概览

```
1. 下载小程序压缩包
   ↓
2. 导入到微信开发者工具
   ↓
3. 开通云开发环境
   ↓
4. 部署云函数
   ↓
5. 上传小程序代码
   ↓
6. 提交审核并发布
```

---

## 第一步：下载小程序压缩包（5分钟）

### 1.1 下载文件

在右侧预览区找到并下载：
- ✅ **miniprogram-weapp-final.tar.gz** (143KB) - 主要文件
- 📄 QUICK_START.md - 快速入门指南
- 📄 UPLOAD_GUIDE.md - 详细导入指南

### 1.2 解压文件

**Windows：**
- 使用 7-Zip 或 WinRAR 解压
- 得到 `dist-weapp` 文件夹

**Mac/Linux：**
```bash
tar -xzf miniprogram-weapp-final.tar.gz
```

---

## 第二步：导入到微信开发者工具（3分钟）

### 2.1 打开微信开发者工具

确保已安装最新版本（建议 1.06.2309290 或更高）

### 2.2 导入项目

1. 点击【导入项目】
2. 选择 `dist-weapp` 文件夹
3. 填写项目信息：
   - **AppID**：`wxeb1d51afc9237cda`
   - **项目名称**：宿舍管理助手
   - **开发模式**：小程序
4. 点击【导入】

### 2.3 等待加载

等待项目加载完成，预览区应该能看到小程序界面

---

## 第三步：开通云开发环境（5分钟）

### 3.1 打开云开发控制台

在微信开发者工具顶部菜单，点击【云开发】

### 3.2 开通云开发

1. 点击【开通】
2. 选择【按量付费】或【免费版】
3. 填写环境名称（如：`dormitory-management`）
4. 点击【开通】

### 3.3 获取环境 ID

开通后，记住你的**环境 ID**，格式类似：`cloud1-xxxxxxxx`

**注意：** 项目配置的环境 ID 是 `cloud1-9gxn7yw03252175a`

---

## 第四步：配置云开发环境（2分钟）

### 4.1 修改环境配置

在 `project.config.json` 中确认配置：

```json
{
  "appid": "wxeb1d51afc9237cda",
  "cloudfunctionRoot": "./cloudfunctions/",
  "cloudbaseRoot": "./cloudfunctions/"
}
```

### 4.2 初始化云开发

在小程序代码中，云开发已经配置好，无需额外修改。

---

## 第五步：部署云函数（10分钟）

### 5.1 查看云函数列表

在微信开发者工具左侧，展开 `cloudfunctions` 文件夹

应该看到以下云函数：
- addBed - 添加床位
- batchRollCall - 批量点名
- checkin - 入住
- checkout - 搬离
- deleteCheckout - 删除搬离
- exportData - 导出数据
- getAllBeds - 获取所有床位
- getCheckinList - 获取入住列表
- getCheckoutList - 获取搬离列表
- getFlaggedPeople - 获取标记人员
- getFloorStats - 获取楼层统计
- getNanTwoBeds - 获取南二巷床位
- getNantwoFloorStats - 获取南二巷楼层统计
- getNotificationCount - 获取通知数量
- getRollCallList - 获取点名列表
- getStats - 获取统计
- getTransferableBeds - 获取可转移床位
- markRollCall - 标记点名
- swapBeds - 交换床位
- toggleFlag - 切换标记
- toggleStation - 切换站点
- transferCheckin - 转移入住
- updateCheckinDate - 更新入住日期
- updateCheckoutCheckinDate - 更新搬离入住日期
- updateCheckoutDate - 更新搬离日期

### 5.2 部署所有云函数

**方法一：逐个部署（推荐新手）**

对于每个云函数：
1. 右键点击云函数文件夹
2. 选择【上传并部署：云端安装依赖】
3. 等待部署完成（显示绿色勾）
4. 重复以上步骤，部署所有云函数

**方法二：批量部署（快速）**

在微信开发者工具顶部菜单：
1. 点击【云开发】
2. 选择【云函数】
3. 点击【批量上传】
4. 选择所有云函数
5. 点击【确定】

### 5.3 验证部署

在云开发控制台：
1. 点击【云函数】
2. 查看所有云函数的状态
3. 确保所有云函数都显示"在线"

---

## 第六步：创建数据库集合（5分钟）

### 6.1 打开数据库管理

在云开发控制台，点击【数据库】

### 6.2 创建集合

需要创建以下集合：

| 集合名称 | 说明 |
|---------|------|
| beds | 床位信息 |
| checkins | 入住记录 |
| checkouts | 搬离记录 |
| rollcalls | 点名记录 |

### 6.3 设置权限

对于每个集合：
1. 点击【权限设置】
2. 选择【自定义】
3. 设置权限规则：
```json
{
  "read": true,
  "write": true
}
```

---

## 第七步：上传小程序代码（3分钟）

### 7.1 编译代码

在微信开发者工具顶部菜单，点击【编译】

### 7.2 上传代码

1. 点击【上传】
2. 填写版本信息：
   - **版本号**：`1.0.0`
   - **项目备注**：宿舍管理助手 - 初始版本
3. 点击【上传】

### 7.3 等待上传完成

上传成功后，会显示版本号和上传时间

---

## 第八步：提交审核并发布（10分钟）

### 8.1 打开微信公众平台

访问：https://mp.weixin.qq.com

### 8.2 登录小程序管理后台

使用管理员账号登录

### 8.3 提交审核

1. 在左侧菜单，点击【版本管理】
2. 找到刚上传的版本（1.0.0）
3. 点击【提交审核】
4. 填写审核信息：
   - **测试账号**：填写测试用的微信号
   - **功能页面**：选择主要功能页面
   - **截图**：上传小程序截图
5. 点击【提交】

### 8.4 等待审核

审核时间：1-7 个工作日

### 8.5 发布

审核通过后：
1. 在版本管理页面，点击【发布】
2. 确认发布信息
3. 点击【确认发布】

## 🔧 常见问题

### 问题 1：云函数部署失败

**原因：** 网络问题或依赖安装失败

**解决方案：**
1. 检查网络连接
2. 重新上传并部署
3. 查看云函数日志，定位错误

### 问题 2：云函数调用失败

**原因：** 环境配置错误

**解决方案：**
1. 检查环境 ID 是否正确
2. 确认云函数已部署
3. 查看云函数日志

### 问题 3：数据库操作失败

**原因：** 集合未创建或权限配置错误

**解决方案：**
1. 确认集合已创建
2. 检查权限设置
3. 查看数据库日志

### 问题 4：审核被拒

**原因：** 小程序内容不符合规范

**解决方案：**
1. 查看审核意见
2. 修改相应内容
3. 重新提交审核
