## 常见问题

### Q1: 解压失败怎么办？

**可能原因：**
- 文件下载不完整
- 解压工具不支持 .tar.gz 格式

**解决方法：**
1. 重新下载压缩包
2. 使用 7-Zip 或 WinRAR 解压
3. 或使用在线解压工具

### Q2: 导入项目时找不到 dist-weapp 文件夹？

**解决方法：**
1. 确认已正确解压压缩包
2. 检查解压后的文件夹名称
3. 使用「浏览」按钮选择文件夹

### Q3: 导入后编译失败？

**检查清单：**
- [ ] AppID 是否正确（wxeb1d51afc9237cda）
- [ ] 项目目录是否选择了 dist-weapp
- [ ] 项目类型是否选择了「小程序」

**解决方法：**
- 重新导入项目
- 确认所有选项正确

### Q4: 上传失败怎么办？

**可能原因：**
- 网络问题
- 微信开发者工具版本过低
- 微信账号权限问题

**解决方法：**
1. 检查网络连接
2. 更新微信开发者工具
3. 确认账号有上传权限
