## 方式二：微信小程序开发调试

小程序无法直接访问 `localhost`，需要以下方案：

### 方案 A：使用开发工具的本地代理

1. 打开微信开发者工具
2. 设置 → 代理设置 → 使用系统代理
3. 构建小程序版本：
   ```bash
   cd /workspace/projects
   PROJECT_DOMAIN=http://localhost:3000 pnpm build:weapp
   ```
4. 导入 `dist/weapp` 目录到开发者工具

### 方案 B：使用内网穿透（推荐）

1. **安装 ngrok**：https://ngrok.com/download

2. **启动内网穿透**：
   ```bash
   ngrok http 3000
   ```

3. **记录生成的公网地址**，例如：
   ```
   https://abc123.ngrok.io
   ```

4. **构建小程序**：
   ```bash
   cd /workspace/projects
   PROJECT_DOMAIN=https://abc123.ngrok.io pnpm build:weapp
   ```

5. **在开发者工具中导入 `dist/weapp` 目录**

## 服务管理命令

### 启动服务

```bash
# 启动后端
cd /workspace/projects/server
pnpm start

# 启动前端开发服务
cd /workspace/projects
coze dev
```

### 停止服务

```bash
# 停止后端
pkill -f "node dist/src/main"

# 停止前端
pkill -f "taro build"
```

### 查看日志

```bash
# 后端日志
tail -f /tmp/server.log

# 前端日志
tail -f /tmp/coze-logs/dev.log
```
