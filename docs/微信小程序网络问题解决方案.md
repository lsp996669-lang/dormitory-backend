# 微信小程序网络问题解决方案

## ⚠️ 重要说明

**微信小程序无法直接访问 localhost**，这是微信的安全限制。即使勾选「不校验合法域名」选项也不行。

---

## 🎯 推荐方案：使用 H5 版本

**这是最简单、最可靠的方式！**

### 访问地址

👉 **http://localhost:5000**

### 为什么推荐 H5？

1. **无需额外配置**：Vite 代理自动转发请求
2. **功能完整**：所有功能与小程序版本一致
3. **实时热更新**：修改代码立即生效
4. **调试方便**：使用浏览器开发者工具

---

## 🔧 备选方案：内网穿透（如果必须使用小程序）

如果您必须在微信小程序中调试，可以使用内网穿透工具。

### 方案一：使用 ngrok（推荐）

#### 1. 注册 ngrok 账号

访问 https://ngrok.com/signup 注册免费账号

#### 2. 获取 authtoken

登录后，在 Dashboard 中找到 authtoken

#### 3. 配置 ngrok

```bash
ngrok config add-authtoken YOUR_AUTHTOKEN
```

#### 4. 启动内网穿透

```bash
ngrok http 3000
```

会显示类似：

```
Forwarding    https://abc123.ngrok.io -> http://localhost:3000
```

#### 5. 记录公网地址

复制 `https://abc123.ngrok.io` 这个地址

#### 6. 重新构建小程序

```bash
cd /workspace/projects
PROJECT_DOMAIN=https://abc123.ngrok.io pnpm build:weapp
```

#### 7. 在微信开发者工具中重新导入

---

### 方案二：使用 localtunnel（免费，无需注册）

```bash
# 安装（已安装）
npm install -g localtunnel

# 启动
lt --port 3000

# 会显示类似：your url is: https://xxx.loca.lt
```

然后按相同步骤重新构建小程序。

⚠️ **注意**：localtunnel 免费版可能不稳定，建议使用 ngrok。

---

## 📊 方案对比

| 方案 | 优点 | 缺点 |
|------|------|------|
| **H5 版本** | 简单、稳定、实时更新 | 需要在浏览器中使用 |
| **ngrok** | 稳定、速度快 | 需要注册账号 |
| **localtunnel** | 免费、无需注册 | 不稳定、可能被墙 |

2. 在浏览器中打开：http://localhost:5000
3. 点击「管理员登录」按钮
4. 使用邀请码 `TEST2024` 验证账号

## ❓ 常见问题

### Q: 为什么小程序不能访问 localhost？

A: 这是微信的安全限制，小程序只能访问：
- HTTPS 域名（需在后台配置）
- 或通过内网穿透工具映射的公网地址

### Q: H5 和小程序功能一样吗？

A: 是的，功能完全一致，只是运行环境不同。

### Q: 内网穿透地址会变吗？

A: 
- ngrok 免费版：每次重启地址会变
- ngrok 付费版：可以使用固定域名
- localtunnel：地址会变

### Q: 可以同时使用 H5 和小程序吗？

A: 可以！它们使用相同的后端服务，数据实时同步。
