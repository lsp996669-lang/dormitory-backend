## ⚠️ 注意事项

### 内网穿透地址有效期

localtunnel 的地址 **每次启动都会变化**。

如果隧道断开，需要重新执行：

```bash
# 重新启动隧道
lt --port 3000

# 重新构建小程序
PROJECT_DOMAIN=新的隧道地址 pnpm build:weapp
```

### 保持服务运行

确保以下服务持续运行：

```bash
# 后端服务（端口 3000）
cd /workspace/projects/server && node dist/src/main.js

# 内网穿透
lt --port 3000
```

## 🔧 常用命令

### 查看当前隧道地址

```bash
cat /tmp/tunnel.log
```

### 重启隧道

```bash
pkill -f "lt --port"
lt --port 3000 > /tmp/tunnel.log 2>&1 &
```

### 查看后端日志

```bash
tail -f /tmp/server.log
```
