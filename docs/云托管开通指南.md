# 微信云托管部署指南

## 问题：找不到云托管入口

您当前的云开发控制台显示的是传统云开发服务（云函数、云数据库、云存储），而**云托管是独立的服务**。

---

## 方法一：直接访问云托管控制台

👉 **打开这个链接**：https://cloud.weixin.qq.com/

1. 使用小程序管理员微信扫码登录
2. 如果是第一次使用，需要点击「立即开通」
3. 选择您的环境或创建新环境

---

## 方法二：从微信公众平台进入

1. 登录微信公众平台：https://mp.weixin.qq.com/
2. 左侧菜单 → 云开发 → 云托管
3. 如果没有显示，说明需要先开通

---

## 开通云托管的条件

1. **小程序必须是企业认证**
2. **需要开通云开发基础版或以上**
3. **需要在微信开放平台注册**

如果您的账号不满足条件，可以考虑其他部署方案。

---

## 方法三：其他部署方案

### 选项 A：使用本地服务器 + 内网穿透

**优点**：免费、快速测试
**缺点**：需要保持电脑开机

1. 在本地运行后端服务：
```bash
cd /workspace/projects/server
pnpm install
pnpm start
```

2. 安装内网穿透工具（ngrok/花生壳）
3. 将本地服务映射到公网地址
4. 在小程序中使用该地址

### 选项 B：部署到云服务器

推荐平台：
- **腾讯云 CloudBase**：https://console.cloud.tencent.com/tcb
- **阿里云**：轻量应用服务器
- **Vercel**：免费部署 Node.js 应用
