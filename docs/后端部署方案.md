# 后端部署方案

## 方案一：开通微信云托管（推荐）

### 步骤 1：访问云托管控制台

云托管不在小程序云开发控制台，需要单独访问：

👉 **https://cloud.weixin.qq.com/**

1. 使用小程序管理员微信扫码登录
2. 选择您的小程序「宿舍管理系统」
3. 如果没有开通云托管，点击「立即开通」

### 步骤 2：创建服务

开通后，创建一个新服务：
- 服务名称：`dormitory-api`
- 部署方式：代码库
- 代码仓库：`https://github.com/lsp996669-lang/dormitory-backend`
- 分支：`main`

### 步骤 3：配置环境变量

```
COZE_SUPABASE_URL=你的Supabase项目URL
COZE_SUPABASE_ANON_KEY=你的Supabase匿名密钥
```

### 步骤 4：发布

点击发布，等待构建完成。

## 方案二：本地开发 + 内网穿透

如果暂时无法开通云托管，可以使用本地服务器 + 内网穿透：

### 步骤 1：启动本地后端服务

```bash
cd /workspace/projects/server

# 安装依赖（如果还没安装）
pnpm install

# 设置环境变量
export COZE_SUPABASE_URL="你的Supabase项目URL"
export COZE_SUPABASE_ANON_KEY="你的Supabase匿名密钥"

# 启动服务
pnpm start
```

### 步骤 2：安装内网穿透工具

推荐使用 ngrok 或 花生壳：

```bash
# 安装 ngrok
# 访问 https://ngrok.com 注册并下载

# 启动内网穿透
ngrok http 3000
```

### 步骤 3：配置前端域名

修改前端配置，将内网穿透地址设置为 `PROJECT_DOMAIN`：

```bash
# 构建 H5 版本
PROJECT_DOMAIN=https://xxx.ngrok.io pnpm build:web

# 构建小程序版本
PROJECT_DOMAIN=https://xxx.ngrok.io pnpm build:weapp
```

## 推荐方案

**正式使用**：方案一（微信云托管）
- 优点：微信生态、稳定可靠、自动扩缩容
- 适合：生产环境

**临时测试**：方案二（本地 + 内网穿透）
- 优点：快速测试、无需额外费用
- 适合：开发调试
