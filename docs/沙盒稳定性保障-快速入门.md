## 🚨 紧急恢复

### 恢复代码

```bash
# 从远程仓库恢复
cd /workspace/projects
git fetch origin
git reset --hard origin/main

# 从备份恢复
tar -xzf backups/backup-20260412-030030.tar.gz
```

### 恢复数据

1. **从 Supabase 自动备份恢复**
   - 登录 Supabase 控制台
   - 选择项目
   - 进入「Database」→「Backups」
   - 选择备份日期
   - 点击「Restore」

2. **从 Excel 备份恢复**
   - 使用小程序的数据导入功能
   - 导入备份数据文件
