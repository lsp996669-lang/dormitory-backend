## 使用指南

### 下载压缩包
1. 从服务器下载 `dormitory-miniapp-v1.0.2.tar.gz`
2. 文件大小：136K

### 解压
```bash
# Windows
使用 WinRAR 或 7-Zip 解压

# macOS / Linux
tar -xzf dormitory-miniapp-v1.0.2.tar.gz
```

### 导入微信开发者工具
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择 `dist-weapp` 文件夹
4. AppID：`wxeb1d51afc9237cda`
5. 项目名称：`宿舍管理助手 v1.0.2`
6. 点击「导入」

### 上传体验版
1. 点击「编译」
2. 点击「上传」
3. 版本号：`1.0.2`
4. 项目备注：
   ```
   v1.0.2 - 新增南二巷2楼201房间3号床

   更新内容：
   - 为南二巷24号宿舍2楼201添加3号床
   - 3号床上铺和下铺均为空闲状态
   - 201房间总床位数从4张增加到6张
   ```
5. 点击「上传」

### 设置体验版
1. 登录微信小程序后台
2. 进入「管理」→「版本管理」
3. 找到 v1.0.2 版本
4. 点击「设为体验版」
