# 真机预览网络问题解决方案

## 问题描述
真机预览时无法连接网络，所有 API 请求失败。

## 根本原因
微信小程序在真机上运行时，会严格检查网络请求的域名是否在**request 合法域名白名单**中。如果未配置，所有网络请求都会被拦截。

---

## 解决方案

### 步骤 1：登录微信小程序后台

访问：[https://mp.weixin.qq.com](https://mp.weixin.qq.com)

使用小程序账号登录。

### 步骤 2：配置 request 合法域名

1. 进入「开发」→「开发管理」
2. 点击「开发设置」
3. 找到「服务器域名」区域
4. 在 **request 合法域名** 中添加：
   ```
   https://server-tan-nu-43.vercel.app
   ```

**注意事项：**
- 域名必须以 `https://` 开头（不支持 http）
- 不要带端口号
- 不要带 `/` 后缀

### 步骤 3：检查域名是否支持 HTTPS

Vercel 默认支持 HTTPS，但建议测试一下：

```bash
curl -I https://server-tan-nu-43.vercel.app/api
```

应返回类似：
```
HTTP/2 200
...
```

### 步骤 4：重新预览

配置完成后，需要：

1. 在微信开发者工具中，点击「清除缓存」→「清除全部缓存」
2. 点击「预览」或「上传代码」
3. 扫码后在真机上打开小程序

## 常见问题

### Q1: 配置后还是无法连接？

**可能原因 1：域名配置未生效**
- 微信小程序后台的域名配置生效时间通常为 5-15 分钟
- 请等待 15 分钟后再试

**可能原因 2：域名未备案**
- 如果域名在国内服务器，需要完成 ICP 备案
- Vercel 域名在海外服务器，不受备案限制

**可能原因 3：CORS 问题**
- 后端需要支持 CORS
- 当前后端已配置 `origin: true`，应无此问题

### Q2: 为什么开发工具可以连接？

开发工具有以下选项可以绕过域名检查：
- 「详情」→「本地设置」→「不校验合法域名...」
- 但此选项仅对开发工具有效，**真机预览无效**

### Q3: 如何确认域名是否配置成功？

在真机上打开小程序，查看调试信息：
- 部分机型会显示「网络请求失败」提示
- 可以在代码中添加日志，查看网络请求的详细信息

### Q4: 可否临时绕过域名检查？

**真机预览无法绕过域名检查**，必须配置合法域名。

## 技术细节

### 后端域名配置
- 域名：`https://server-tan-nu-43.vercel.app`
- 所有 API 请求会自动添加 `/api` 前缀
- 示例：`/api/checkin/list` → `https://server-tan-nu-43.vercel.app/api/checkin/list`

### 前端网络请求
使用 `Network.request()` 封装，自动处理域名拼接：
```typescript
await Network.request({
  url: '/api/checkin/list'
})
// 实际请求：https://server-tan-nu-43.vercel.app/api/checkin/list
```
