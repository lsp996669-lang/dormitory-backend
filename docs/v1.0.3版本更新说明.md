## 🚀 使用指南

### 标记红名

1. 进入入住页面
2. 点击已入住人员的床位
3. 进入人员详情页
4. 点击「标记红名」按钮
5. 确认标记

**效果：**
- 床位边框变为红色
- 姓名变为红色
- 显示红色 "!" 徽章

### 取消标记

1. 进入入住页面
2. 点击已标记人员的床位
3. 进入人员详情页
4. 点击「取消标记」按钮
5. 确认取消

**效果：**
- 床位边框恢复绿色
- 姓名恢复蓝色
- 移除红色 "!" 徽章

### 搬离人员

1. 进入入住页面
2. 点击已入住人员的床位
3. 进入人员详情页
4. 点击「搬离」按钮
5. 输入密码确认

**效果：**
- 人员搬离，床位变为空闲
- 红框自动移除
- 所有标记数据清除

## ⚠️ 注意事项

1. **红框显示：**
   - 只有入住人员被标记后，床位才会显示红框
   - 空床位和维修中床位不受影响

2. **红框移除：**
   - 搬离人员后，红框自动移除
   - 取消标记后，红框自动移除

3. **视觉提示：**
   - 红框 + 红色姓名 + "!" 徽章，三重提示
