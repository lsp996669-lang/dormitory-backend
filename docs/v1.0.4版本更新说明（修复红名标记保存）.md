## 🐛 修复问题

### 问题描述

**问题：** 标记红名后，退出页面，标记就取消了

**根本原因：**
1. 后端查询床位数据时，没有查询 `is_flagged` 字段
2. 前端代码中字段名不一致：
   - 后端返回：`is_flagged`（snake_case）
   - 前端使用：`isFlagged`（camelCase）
3. TypeScript 类型定义也使用了 camelCase，导致无法正确读取后端返回的数据

### 2. 前端修复

**文件：** `src/pages/checkin/index.tsx`

#### 2.1 修改 TypeScript 类型定义

```typescript
// 修改前
interface BedInfo {
  checkIn?: {
    id: number
    name: string
    idCard: string
    phone: string
    checkInTime: string
    isStationMarked?: boolean
    isRider?: boolean
    isFlagged?: boolean
    stationName?: string | null
  }
}

// 修改后
interface BedInfo {
  checkIn?: {
    id: number
    name: string
    id_card: string
    phone: string
    check_in_time: string
    is_station_marked?: boolean
    is_rider?: boolean
    is_flagged?: boolean
    station_name?: string | null
  }
}
```

#### 2.2 修改字段名使用方式

所有使用 `bed.checkIn.isFlagged` 的地方改为 `bed.checkIn.is_flagged`

所有使用 `bed.checkIn.checkInTime` 的地方改为 `bed.checkIn.check_in_time`

所有使用 `bed.checkIn.stationName` 的地方改为 `bed.checkIn.station_name`

所有使用 `bed.checkIn.idCard` 的地方改为 `bed.checkIn.id_card`

所有使用 `bed.checkIn.isStationMarked` 的地方改为 `bed.checkIn.is_station_marked`

所有使用 `bed.checkIn.isRider` 的地方改为 `bed.checkIn.is_rider`
