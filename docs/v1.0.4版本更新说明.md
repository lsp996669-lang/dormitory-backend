## 🐛 问题修复

### 问题：沙盒链接断开

**问题描述：**
在使用小程序的「数据导入」功能时，经常出现沙盒链接断开的问题：

```
Image: [https://code.coze.cn/api/sandbox/coze_coding/file/proxy?expire_time=-1&file_path=assets%2Fimage.png&nonce=...&project_id=...&sign=...]
```

**问题原因：**
代码中使用了 Coze 临时文件代理链接，这种链接存在以下问题：
1. **临时性**：即使设置了 `expire_time=-1`（永不过期），仍然会因为沙盒会话过期、签名验证失败等原因失效
2. **不稳定性**：依赖沙盒环境，沙盒重启后链接失效
3. **硬编码问题**：链接直接写在代码中，无法动态更新

**原始代码（有问题）：**
```typescript
// src/pages/import/index.tsx

const fileUrl = 'https://code.coze.cn/api/sandbox/coze_coding/file/proxy?expire_time=-1&file_path=assets%2F2_%E5%8D%97%E5%9B%9B%E5%B7%B7%E4%BD%8F%E5%AE%BF%E4%BA%BA%E5%91%98%E5%90%8D%E5%8D%953%E6%9C%8821%E5%8F%B7%283%29.xlsx&nonce=...&project_id=...&sign=...'
```

## 🚀 使用指南

### 数据导入步骤

1. 打开小程序
2. 点击「数据导入」
3. 点击「上传Excel文件」
4. 选择本地 Excel 文件
5. 确认导入

### Excel 文件格式

- 支持 xlsx 和 xls 格式
- 每个工作表代表一个楼层
- 自动识别入住人员和搬离人员
- 导入前会清空现有数据

---

## ⚠️ 注意事项

1. **需要准备 Excel 文件**
   - v1.0.4 版本不再提供预设文件
   - 需要用户自己准备 Excel 文件
   - 建议保留一份 Excel 文件作为备份

2. **导入前备份**
   - 导入前会清空现有数据
   - 建议先导出当前数据作为备份
   - 确认无误后再导入

3. **文件格式**
   - 确保 Excel 文件格式正确
   - 支持的格式：xlsx、xls
   - 不支持的格式：csv、txt
