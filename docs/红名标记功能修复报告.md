### 3. 检查数据库查询

**查询床位数据：**
```bash
curl -s "http://localhost:3000/api/beds/floor/2"
```

**返回的数据结构：**
```json
{
  "id": 542,
  "bed_id": 32,
  "name": "伊木然江·努尔麦麦提",
  "id_card": "652822200808292934",
  "phone": "13657555888",
  "check_in_time": "2025-12-24 08:00:00",
  "is_station_marked": false,
  "is_rider": false,
  "station_name": null
}
```

**问题发现：**
- ❌ 数据中缺少 `is_flagged` 字段
- ❌ 只有 `is_station_marked` 字段
- ❌ 前端无法获取标记状态

---

### 4. 定位根本原因

**检查 beds.service.ts 中的查询语句：**

**第 154 行（getBedsByFloor 方法）：**
```typescript
const { data: checkIns, error: checkInError } = await client
  .from('check_ins')
  .select('id, bed_id, name, id_card, phone, check_in_time, is_station_marked, is_rider, station_name')
  .in('bed_id', bedIds);
```

**问题：**
- ❌ 查询语句中缺少 `is_flagged` 字段
- ❌ 导致前端无法获取红名标记状态

---

## 🔧 修复方案

### 修复内容

在所有查询入住记录的地方，添加 `is_flagged` 字段。

---

### 1. 修复 getBedsByFloor 方法（第 154 行）

**修复前：**
```typescript

  .from('check_ins')
  .select('id, bed_id, name, id_card, phone, check_in_time, is_station_marked, is_rider, station_name')
  .in('bed_id', bedIds);
```

**修复后：**
```typescript
const { data: checkIns, error: checkInError } = await client

```

**改进：**
