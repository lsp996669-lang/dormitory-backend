# ⚡ 30 秒解决"只显示2个床"问题

## 问题
南二巷201房配置了3个床，但只显示2个床。

## 原因
数据库中缺少3号床的记录。

---

## 🚀 快速解决（30 秒）

### 步骤 1：打开小程序测试页面

1. 编译项目或扫码预览
2. 进入「云开发测试页面」

### 步骤 2：添加3号床

1. 找到「添加床位测试（南二巷201房）」
2. 床位编号输入：`3`
3. 点击「添加3号床位（上下铺）」

### 步骤 3：刷新页面

1. 返回楼层页面
2. 下拉刷新
3. ✅ 完成！现在显示3个床了

---

## 📊 添加前后对比

| 操作前 | 操作后 |
|--------|--------|
| ❌ 只显示 1 号床、2 号床 | ✅ 显示 1 号床、2 号床、3 号床 |

---

## 🆘 还是不行？

### 方法 2：云开发控制台添加（1 分钟）

1. 打开 https://console.cloud.tencent.com/tcb
2. 进入 `beds` 集合
3. 添加上铺：

```json
{
  "dormitory": "nantwo",
  "floor": "2",
  "room": "201",
  "bed_number": "3",
  "position": "upper",
  "status": "empty"
}
```

4. 添加下铺：

```json
{
  "dormitory": "nantwo",
  "floor": "2",
  "room": "201",
  "bed_number": "3",
  "position": "lower",
  "status": "empty"
}
```

5. 刷新小程序
