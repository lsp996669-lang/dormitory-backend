快速导入步骤
==========================================

1. 解压此文件，得到 dist-weapp 文件夹

2. 打开微信开发者工具
   下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

3. 点击"导入项目"，选择 dist-weapp 文件夹

4. 填写信息：
   - 项目名称: 宿舍管理助手
   - AppID: wxeb1d51afc9237cda
   - 后端服务: 不使用云服务

5. 点击"导入"

6. 等待加载完成后，点击顶部菜单的"上传"上传体验版

==========================================
功能说明
==========================================

✅ 楼层统计 - 查看南四巷和南二巷宿舍楼层信息
✅ 入入登记 - 登记人员入住信息
✅ 床位详情 - 查看床位详细信息
✅ 导出数据 - 导出入住和搬离数据到Excel
⚠️ 其他功能 - 部分功能需要云函数支持
