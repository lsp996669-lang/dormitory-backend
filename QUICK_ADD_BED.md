# ⚡ 1 分钟添加南二巷201房3号床

## 🚀 最简单的方法（推荐）

### 方法 1：使用小程序测试页面（30 秒）

1. **打开小程序**
   - 编译项目或扫码预览

2. **进入测试页面**
   - 找到「云开发测试页面」
   - 在「添加床位测试」区域
   - 床位编号输入：`3`
   - 点击「添加3号床位（上下铺）」

3. **完成！**
   - 看到 ✅ 成功提示即完成

---

### 方法 2：云开发控制台添加（1 分钟）

1. **打开云开发控制台**
   - https://console.cloud.tencent.com/tcb

2. **测试云函数**
   - 进入「云函数」
   - 选择 `addBed`
   - 点击「云端测试」
   - 输入：
     ```json
     {
       "dormitory": "nantwo",
       "floor": 2,
       "room": "201",
       "bedNumber": 3
     }
     ```
   - 点击「执行」

3. **完成！**
   - 看到 `code: 200` 即成功

---

### 方法 3：数据库手动添加（2 分钟）

1. **打开云开发控制台**
   - https://console.cloud.tencent.com/tcb

2. **进入数据库**
   - 选择 `beds` 集合

3. **添加上铺**
   ```json
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3",
     "position": "upper",
     "status": "empty"
   }
   ```

4. **添加下铺**
   ```json
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3",
     "position": "lower",
     "status": "empty"
   }
   ```

5. **完成！**

## 📝 常见问题

**A**：3号床已经存在，无需重复添加

### Q：提示"云函数调用失败"
