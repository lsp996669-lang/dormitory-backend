# 🛏️ 添加南二巷201房3号床

## 目标
在南二巷24号宿舍2楼201房间添加3号床（上下铺）。

---

## 📋 操作步骤

### 方法 1：使用测试页面添加（推荐）

#### 前提条件
- ✅ 云开发已配置完成
- ✅ 云函数已上传
- ✅ 数据库集合已创建

#### 步骤

1. **打开小程序**
   - 在微信开发者工具中编译项目
   - 或扫码预览小程序

2. **进入测试页面**
   - 进入「云开发测试页面」（pages/test-cloud/index）

3. **添加3号床**
   - 在「添加床位测试（南二巷201房）」区域
   - 床位编号输入：`3`
   - 点击「添加3号床位（上下铺）」按钮

4. **查看结果**
   - 成功提示：✅ 成功：成功添加床位：201房3号床（上下铺）
   - 失败提示：❌ 失败：该床位已存在

---

### 方法 2：在云开发控制台直接添加

#### 步骤

1. **打开云开发控制台**
   - 访问：https://console.cloud.tencent.com/tcb

2. **进入数据库**
   - 选择您的环境
   - 点击「数据库」
   - 选择 `beds` 集合

3. **添加上铺记录**

   点击「添加记录」，输入以下 JSON：

   ```json
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3",
     "position": "upper",
     "status": "empty",
     "created_at": "2025-01-08T00:00:00.000Z",
     "updated_at": "2025-01-08T00:00:00.000Z"
   }
   ```

4. **添加下铺记录**

   再次点击「添加记录」，输入以下 JSON：

   ```json
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3",
     "position": "lower",
     "status": "empty",
     "created_at": "2025-01-08T00:00:00.000Z",
     "updated_at": "2025-01-08T00:00:00.000Z"
   }
   ```

5. **验证添加成功**

   在数据库中查询：
   ```json
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3"
   }
   ```

   应该能看到2条记录（上铺和下铺）。

---

### 方法 3：调用云函数（开发方式）

#### 步骤

1. **在云开发控制台测试云函数**

   - 访问：https://console.cloud.tencent.com/tcb
   - 进入「云函数」
   - 选择 `addBed` 云函数
   - 点击「云端测试」

2. **输入测试参数**

   ```json
   {
     "dormitory": "nantwo",
     "floor": 2,
     "room": "201",
     "bedNumber": 3
   }
   ```

3. **点击「执行」**

4. **查看结果**

   成功响应：
   ```json
   {
     "code": 200,
     "msg": "成功添加床位：201房3号床（上下铺）",
     "data": {
       "dormitory": "nantwo",
       "floor": 2,
       "room": "201",
       "bedNumber": 3,
       "positions": ["upper", "lower"]
     }
   }
   ```

## 🆘 常见问题

### Q：添加后看不到3号床怎么办？

**A**：
1. 刷新小程序页面
2. 重新进入房间页面
3. 检查数据库是否有记录

### Q：想删除3号床怎么办？

**A**：
1. 在云开发控制台数据库中
2. 找到3号床的记录（上铺和下铺）
3. 点击删除

### Q：想修改床位状态怎么办？

**A**：
1. 在云开发控制台数据库中
2. 找到对应的床位记录
3. 修改 `status` 字段：
   - `empty` - 空闲
   - `occupied` - 已占用
   - `maintenance` - 维修中
