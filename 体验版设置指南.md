## 🚀 设置体验版完整步骤

### 第一步：上传代码到微信平台

#### 方式一：使用微信开发者工具（推荐）

1. **下载微信开发者工具**
   - 下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
   - 选择对应系统版本（Windows/Mac/Linux）

2. **导入项目**
   ```
   项目目录: /workspace/projects/dist-weapp
   AppID: wxeb1d51afc9237cda
   项目名称: 宿舍管理助手
   ```

3. **预览检查**
   - 点击"预览"按钮，扫码检查功能是否正常
   - 确认搜索功能、入住、搬离等核心功能正常

4. **上传代码**
   - 点击右上角"上传"按钮
   - 填写版本号: `1.0.0`
   - 填写项目备注: `宿舍管理助手 - 首个正式版本，支持搜索、入住、搬离等功能`
   - 点击"上传"按钮

---

### 第二步：登录微信公众平台

1. **访问微信公众平台**
   - 网址: https://mp.weixin.qq.com
   - 使用管理员微信扫码登录

2. **进入小程序管理后台**
   - 确认登录的是 AppID 为 `wxeb1d51afc9237cda` 的小程序

---

### 第三步：设置为体验版

1. **进入版本管理**
   ```
   左侧菜单: 管理 → 版本管理
   ```

2. **找到开发版本**
   - 在"开发版本"列表中找到刚上传的版本
   - 版本号: 1.0.0
   - 上传时间: 刚刚上传的时间

3. **设为体验版**
   - 点击版本右侧的"⋮"更多按钮
   - 选择"设为体验版"
   - 在弹窗中确认操作

4. **生成体验版二维码**
   - 设为体验版后，会自动生成体验版二维码
   - 点击二维码可放大查看
   - **注意**: 只有体验者才能扫码访问

---

### 第四步：添加体验者

#### 方式一：通过微信号添加（推荐）

1. **进入成员管理**
   ```
   左侧菜单: 管理 → 成员管理 → 项目成员
   ```

2. **添加体验者**
   - 点击"添加成员"按钮
   - 选择"体验者"角色
   - 输入体验者的微信号
   - 点击"确认添加"

3. **批量添加（可选）**
   - 最多可添加 100 名体验者
   - 建议添加内部测试人员

#### 方式二：通过邀请链接

1. **生成邀请链接**
   - 在"成员管理"页面
   - 点击"邀请成员"
   - 选择"体验者"角色
   - 复制邀请链接

2. **分享链接**
   - 将链接发送给测试人员
   - 测试人员点击链接即可成为体验者

---

### 第五步：扫码体验

1. **获取体验版二维码**
   - 返回"版本管理"页面
   - 在"体验版"区域查看二维码

2. **扫码体验**
   - 使用微信扫描二维码
   - 体验者会自动进入小程序

3. **测试核心功能**

## ⚠️ 常见问题

### 问题1: 找不到上传的版本
**解决方案：**
- 确认已成功上传代码
- 刷新页面重新查看
- 检查是否登录了正确的小程序账号（AppID: wxeb1d51afc9237cda）

### 问题2: 扫码后提示"不是体验者"
**解决方案：**
- 确认已在"成员管理"中添加该微信号为体验者
- 体验者需要先关注小程序或通过邀请链接加入
- 重新扫码尝试

### 问题3: 体验版无法访问后端服务
**解决方案：**
- 检查服务器域名是否已添加到白名单
- 确认云托管服务正在运行
- 查看开发者工具控制台的错误信息

### 问题4: 搜索功能无响应
**解决方案：**
- 确认后端服务正常运行
- 检查网络连接
- 查看控制台日志
