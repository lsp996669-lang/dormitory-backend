# 方案一：本地后端 + 内网穿透（最快方案）

## ⏱️ 预计耗时：5分钟

## 📋 操作步骤

### 步骤1：安装内网穿透工具

**推荐工具：ngrok（免费）**

**Windows/Mac/Linux 通用：**

```bash
# 安装 ngrok
npm install -g ngrok

# 或者下载安装包
# https://ngrok.com/download
```

---

### 步骤2：启动内网穿透

**在终端执行：**

```bash
# 穿透本地 3000 端口
ngrok http 3000
```

**执行后会显示：**
```
Session Status                online
Account                       你的账号
Version                       3.x.x
Region                        Asia Pacific (ap)
Latency                       45ms
Web Interface                 http://127.0.0.1:4040
Forwarding                    https://abc123.ngrok.io -> http://localhost:3000
                                          ↑
                                    这是你的公网地址
```

**复制这个 https 地址，例如：**
```
https://abc123.ngrok.io
```

---

### 步骤3：修改小程序配置

**修改配置文件：**

打开 `/workspace/projects/.env.local` 文件：

```bash
# 修改前
PROJECT_DOMAIN=https://dormitory-api-xxx.sh.run.tcloudbase.com

# 修改后（使用 ngrok 提供的地址）
PROJECT_DOMAIN=https://abc123.ngrok.io
```

---

### 步骤4：重新构建小程序

```bash
cd /workspace/projects
pnpm build:weapp
```

---

### 步骤5：上传小程序

```bash
miniprogram-ci upload \
  --pp ./dist-weapp \
  --pkp ./private.wx.key \
  --appid wxeb1d51afc9237cda \
  -r 3 \
  --uv 1.0.2 \
  -d "使用内网穿透后端"
```

---

### 步骤6：配置域名白名单

**在微信公众平台添加域名：**

1. 登录 https://mp.weixin.qq.com
2. 进入"开发" → "开发管理" → "开发设置"
3. 在"服务器域名"中添加：
```
request 合法域名:
https://abc123.ngrok.io
```

⚠️ **注意：**
- ngrok 免费版域名会变化
- 每次重启 ngrok 都需要更新配置
- 适合临时测试使用

## 💡 替代工具

### 工具1：localtunnel（更简单）

```bash
# 安装
npm install -g localtunnel

# 启动
lt --port 3000

# 会得到类似这样的地址
your url is: https://xxx-xxx-xxx.loca.lt
```

### 工具2：Cloudflare Tunnel（稳定）

```bash
# 安装
npm install -g cloudflared

# 启动
cloudflared tunnel --url http://localhost:3000

# 会得到类似这样的地址
Your quick Tunnel has been created! Visit it at:
https://xxx-xxx.trycloudflare.com
```

## ⚠️ 注意事项

1. **保持后端运行**
   - 确保 `coze dev` 在运行
   - 确保 ngrok 在运行
   - 电脑不能关机或休眠

2. **域名变化**
   - ngrok 免费版每次启动域名都会变
   - 需要重新修改配置、重新构建
   - 建议用于临时测试

3. **安全性**
   - 不要在生产环境使用
   - 仅用于开发测试
