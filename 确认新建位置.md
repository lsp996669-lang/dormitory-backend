## 📋 正确的操作流程

### 步骤1：确认按钮
**您应该点击包含以下关键词的按钮：**
- "新建服务"
- "创建服务"
- "新建版本"

### 步骤2：选择部署方式
**在弹出的界面中，选择：**
```
✅ 本地上传
✅ 代码包上传
✅ 自定义创建

❌ 不要选择：
- 从模板创建
- 从镜像创建
```

### 步骤3：填写信息
```
服务名称: dormitory-api
服务介绍: 宿舍管理后端服务
```

### 步骤4：上传部署包
```
选择文件: server-deploy.tar.gz (26KB)
位置: /workspace/projects/server-deploy.tar.gz
```

### 步骤5：配置环境变量（关键步骤）
```bash
SUPABASE_URL=https://你的项目ID.supabase.co
SUPABASE_ANON_KEY=你的匿名密钥
NODE_ENV=production
PORT=3000
```

### 步骤6：配置规格
```
CPU: 0.5核
内存: 512MB
实例: 最小1，最大2
端口: 3000
```

## 🎯 判断标准
