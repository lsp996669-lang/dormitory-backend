## 📱 导入步骤

### 1. 解压文件

下载后，解压文件到本地：

**Windows**：
- 使用 7-Zip 或 WinRAR 解压 `.tar.gz` 文件
- 解压后会得到 `dist-weapp` 文件夹

**Mac/Linux**：
```bash
tar -xzf miniprogram-upload.tar.gz
```

### 2. 打开微信开发者工具

1. 如果没有安装，先下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 打开微信开发者工具
3. 使用微信扫码登录

### 3. 导入项目

1. 点击首页的「**导入项目**」
2. 填写以下信息：

| 项目 | 内容 |
|------|------|
| **项目名称** | 宿舍管理助手 |
| **目录** | 选择解压后的 `dist-weapp` 文件夹 |
| **AppID** | `wxeb1d51afc9237cda` |
| **后端服务** | 不使用云服务（已迁移到云函数） |

3. 点击「**导入**」

### 4. 等待加载完成

- 微信开发者工具会自动加载项目
- 首次加载可能需要几秒钟
- 加载成功后会看到项目界面

## ⚙️ 配置云函数（重要）

由于小程序使用了云函数，需要单独配置：

### 方法 1：在微信开发者工具中配置

1. 在微信开发者工具左侧，点击「**云开发**」
2. 如果提示开通，点击「开通」
3. 选择环境：`cloud1-9gxn7yw03252175a`
4. 点击「**确定**」

### 方法 2：上传云函数

1. 在微信开发者工具中，找到 `cloudfunctions` 文件夹
2. **注意**：压缩包中没有云函数，需要单独配置
3. 右键点击 `cloudfunctions` → 选择「上传并部署：云端安装依赖」
4. 等待部署完成

**或者使用测试数据**：
- 如果暂时不需要所有功能，可以先使用已有的云函数
- 在微信小程序管理后台中，云函数应该已经部署

## 🔧 常见问题

### Q1: 导入时提示"AppID 不存在"

**A**: 检查 AppID 是否正确：
- 正确：`wxeb1d51afc9237cda`
- 确保是您自己的小程序 AppID

### Q2: 导入后提示"缺少云函数"

**A**: 这是因为压缩包中没有包含云函数。您需要：
1. 在微信小程序管理后台配置云函数
2. 或者在微信开发者工具中手动上传云函数

### Q3: 上传体验版失败

**A**: 检查以下几点：
- 确保微信开发者工具已登录
- 检查网络连接
- 确保小程序未违规（首次上传需要）

### Q4: 功能无法使用

**A**: 可能的原因：
1. 云函数未部署 → 需要配置云函数
2. 登录权限问题 → 需要先登录
3. 数据库权限问题 → 检查云开发权限设置
