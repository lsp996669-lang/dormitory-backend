# 🚀 快速部署小程序

## 3 步获取部署包

### 1️⃣ 打开页面
```javascript
wx.navigateTo({ url: '/pages/deploy-download/index' })
```

### 2️⃣ 生成下载
点击「生成部署包」按钮

### 3️⃣ 下载并部署
- 复制链接到浏览器下载
- 解压后用微信开发者工具打开

## 📱 部署步骤

1. 打开微信开发者工具
2. 导入项目 → 选择 `dist-weapp` 目录
3. AppID：`wxeb1d51afc9237cda`
4. 点击「预览」生成二维码测试

## 📦 文件信息

- 大小：144 KB
- 格式：.tar.gz
- 有效期：2 小时
