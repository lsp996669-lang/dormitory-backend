## 🚀 还需要做 3 件事

### 1️⃣ 配置合法域名（5 分钟）

**在微信公众平台**：
1. 登录：https://mp.weixin.qq.com
2. 开发 → 开发管理 → 开发设置
3. 服务器域名 → request 合法域名
4. 添加：`https://server-tan-nu-43.vercel.app`
5. 保存

---

### 2️⃣ 开发者工具设置（1 分钟）

**在微信开发者工具中**：
1. 点击右上角「详情」
2. 本地设置
3. **取消勾选**「不校验合法域名」
