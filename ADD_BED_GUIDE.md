# 为南二巷24号201房添加床位说明

## 需求
为南二巷24号宿舍的201房间添加一个上下铺的床位（3号床）。

## 当前状态
- 南二巷24号宿舍 2楼 201房间
- 已有床位：1号床（上铺+下铺）、2号床（上铺+下铺）
- 需要添加：3号床（上铺+下铺）

## 操作步骤

### 步骤1：上传 addBed 云函数

在微信开发者工具中：

1. 右键点击 `cloudfunctions/addBed` 目录
2. 选择「上传并部署：云端安装依赖」
3. 等待上传完成

### 步骤2：调用云函数添加床位

**方法A：使用测试页面（推荐）**

1. 在微信开发者工具中编译项目
2. 访问测试页面：`pages/test-cloud/index`
3. 找到"添加床位测试（南二巷201房）"部分
4. 确认床位编号为 `3`
5. 点击"添加3号床位（上下铺）"按钮
6. 等待提示"添加成功"

**方法B：手动调用云函数**

```typescript
import { Cloud } from '@/cloud'

const res = await Cloud.callFunction('addBed', {
  dormitory: 'nantwo',
  floor: 2,
  room: '201',
  bedNumber: 3
})

if (res.result?.code === 200) {
  console.log('添加成功', res.result.msg)
  // 输出：成功添加床位：201房3号床（上下铺）
} else {
  console.error('添加失败', res.result?.msg)
}
```

### 步骤3：验证添加结果

1. 打开云开发控制台
2. 进入「数据库」→「beds」集合
3. 查询条件：
   ```javascript
   {
     "dormitory": "nantwo",
     "floor": "2",
     "room": "201",
     "bed_number": "3"
   }
   ```
4. 应该看到2条记录：
   - 3号床 上铺（position: 'upper'）
   - 3号床 下铺（position: 'lower'）

### 步骤4：在前端验证

1. 在小程序中进入"楼层"页面
2. 选择"南二巷24号"→"2楼"→"201房"
3. 应该能看到新增的3号床位（包含上铺和下铺）
4. 3号床位状态应该是"空闲"（可点击进行入住登记）

## 常见问题

### Q1: 添加时提示"该床位已存在"

**原因**：3号床位已经存在于数据库中

**解决方案**：
- 检查云数据库中是否已有3号床位的记录
- 如果已有记录，说明之前已经添加过，无需重复添加

### Q2: 云函数上传失败

**原因**：云函数依赖安装失败

**解决方案**：
- 检查网络连接
- 尝试重新上传
- 确认云开发环境已开通

### Q3: 添加成功但前端看不到

**原因**：前端数据未刷新

**解决方案**：
- 下拉刷新页面
- 重新进入房间页面
- 检查云数据库中是否正确添加了记录

### Q4: 如何删除错误的床位

**方法A：通过云开发控制台**
1. 打开云开发控制台
2. 进入「数据库」→「beds」集合
3. 找到要删除的记录
4. 点击「删除」按钮

**方法B：调用云函数（需要开发删除功能）**
- 可以创建 `deleteBed` 云函数来删除指定床位
