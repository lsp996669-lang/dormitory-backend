## 🆘 常见问题

### Q1：云函数上传失败

**错误提示**：「上传失败，环境未初始化」

**解决方法**：
1. 确认云开发环境已关联
2. 在项目详情的「云开发」标签页中检查配置
3. 重新编译项目
4. 重新上传云函数

### Q2：数据库集合创建失败

**错误提示**：「集合已存在」或「权限不足」

**解决方法**：
- 如果集合已存在，跳过创建步骤
- 如果权限不足，在云开发控制台中修改集合权限

### Q3：功能测试失败

**错误提示**：「云函数调用失败」

**解决方法**：
1. 检查云函数是否全部上传成功
2. 检查数据库集合是否已创建
3. 检查云开发环境 ID 是否正确
4. 查看云函数日志，排查错误

### Q4：提示"云开发未初始化"

**错误提示**：「cloud.init:fail」

**解决方法**：
1. 确认 `src/app.tsx` 中的环境 ID 为 `cloud1-9gxn7yw03252175a`
2. 重新编译项目
3. 检查网络连接
4. 重新测试

## 📱 快速测试命令

```javascript
// 测试云开发是否初始化
console.log('环境 ID:', wx.cloud.getWXContext().ENV);

// 测试云函数调用
wx.cloud.callFunction({
  name: 'getCheckinList',
  data: { location: '南四巷180号' },
  success: res => console.log('✅ 云函数调用成功', res),
  fail: err => console.error('❌ 云函数调用失败', err)
});
```
