# 方案二：微信开发者工具本地调试（最简单）

## ⏱️ 预计耗时：2分钟

## 📋 操作步骤

### 步骤1：修改配置指向本地后端

**修改配置文件：**

打开 `/workspace/projects/.env.local` 文件：

```bash
# 修改为本地后端
PROJECT_DOMAIN=http://localhost:3000
```

---

### 步骤2：重新构建小程序

```bash
cd /workspace/projects
pnpm build:weapp
```

---

### 步骤3：使用微信开发者工具

**打开微信开发者工具：**
```
1. 导入项目
   项目目录: /workspace/projects/dist-weapp
   AppID: wxeb1d51afc9237cda
   
2. 点击右上角"详情"
   
3. 勾选"不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书"
   ↑ 这一步很重要！
```

---

### 步骤4：预览和调试

**在开发者工具中：**
1. 点击"预览"按钮
2. 扫码在手机上体验
3. 或直接在模拟器中调试

---

## ✅ 验证搜索功能

**在模拟器或真机调试中：**
1. 登录小程序
2. 输入"138"或"张"
3. 点击搜索
4. 应该能看到搜索结果

---

## 💡 关键设置

### 重要：关闭域名校验

**在微信开发者工具中：**

```
右上角"详情"
↓
"本地设置"标签
↓
勾选：不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
```

**⚠️ 不勾选的话会报错：**
```
request:fail url not in domain list
```

---

## 🎯 优缺点

**优点：**
- ✅ 2分钟内可用
- ✅ 无需任何部署
- ✅ 完整的调试功能
- ✅ 可以看到控制台日志
- ✅ 可以断点调试

**缺点：**
- ❌ 仅在开发环境可用
- ❌ 需要使用开发者工具
- ❌ 无法让其他用户体验
- ❌ 真机调试需要扫码

---

## 📱 真机调试步骤

### 方式1：预览码

```
开发者工具中点击"预览"
↓
生成二维码
↓
手机微信扫码体验
```

### 方式2：真机调试

```
开发者工具中点击"真机调试"
↓
生成调试二维码
↓
手机微信扫码
↓
可以在手机上看到vConsole日志
```

---

## 🔍 调试搜索功能

**如果搜索失败：**

1. **打开控制台**
   - 开发者工具底部"Console"标签
   - 查看是否有错误信息

2. **检查网络请求**
   - 点击"Network"标签
   - 查看请求是否发送成功
   - 查看返回的数据

3. **查看日志输出**
   - 搜索时会打印：
   ```
   [Floor] 搜索结果: {...}
   ```
